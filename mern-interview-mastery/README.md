## 🚀 How to Run the Machine Coding Tasks

Each machine coding task is an independent project bootstrapped using Vite and React 19.

To run any of the tasks:

1. **Navigate** into the chosen task directory:
   ```bash
   cd machine-coding-tasks/08-image-upload
   ```

2. **Install** node modules:
   ```bash
   npm install
   ```

3. **Start** the development server:
   ```bash
   npm run dev
   ```
